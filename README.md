# Flight Tracking & Simulation Dashboard
 
A real-time flight monitoring and simulation dashboard built using Angular 20, RxJS, Angular Material, and Leaflet.
 
---
 
## Features
 
### Flight Monitoring Dashboard
- Real-time flight tracking
- Live flight simulation
- Dynamic status updates
- Search and filtering
 
### Interactive Flight Map
- Aircraft movement visualization
- Route path rendering
- Departure and destination markers
- Live coordinate updates
 
### Flight Details View
- Aircraft information
- Route details
- Timeline
- Communication logs
 
### Alert System
- Altitude drop alerts
- Speed threshold alerts
- Alert counter panel
 
### Incident Reporting
- Reactive Forms
- Dynamic validations
- Severity-based validations
 
### Role-Based Access
- Supervisor
- Controller
- Viewer
 
---
